Введение приказа в действие

Для введения приказа в действие необходимо принять два решения:

  • Вид печатной формы - печатная форма может быть либо табличной, либо текстовой. При введении приказа в действие должна быть зафиксирована одна из них - какая конкретно - для каждого случае решение принимается отдельно.
  • Дата введения в действие - по умолчанию подставляется текущая дата, если это не так - то можно указать любую другую дату введения в действие.
