Создание приказа

Выберите тип создаваемого проекта приказа из списка. В зависимости от создаваемого типа приказа, дальнейшие действия могут быть следующими:

  • Переводной приказ - создает только по целому курсу; следующим шагом будет предложено выбрать курс; в списки студентов приказа автоматически попадут все обучающиеся на этом курсе того факультета, по которому издается приказ; дальнейшая задача по созданию проекта приказа состоит в том, чтобы определить, какие из обучающихся переводятся без задолженностей, а какие - с задолженностями, разместить их по соответствующим параграфам (см. редактирование приказа) и для перевода обучающихся с задолженностями - ввести в качестве их атрибутов эти задолженности и срок сдачи.
  • Об отчислении, о предоставлении академических отпусков, о допуске к учебному процессу, о восстановлении в МГЛУ, о переносе срока ликвидации академических задолженностей, о признании факта ликвидации академических задолженностей, об изменении фамилии, имени или отчества обучающегося, о переводе с одной образовательной программы на другую, о переходе с платной основы обучения на бюджетную - приказы, при создании проекта которых создается пустой проект без включенных в приказ обучающихся; нужных обучающихся следует добавить туда посредством поиска или навигации.
  • О зачислении в МГЛУ, о зачислении в порядке перевода из другого вуза - приказ специфичен тем, что работает со студентами, которых еще нет в базе данных и поэтому их нельзя просто добавить в приказ через поиск или навигацию; в этих приказах студенты добавляются в 2 этапа: сначала с помощью формы "добавить студента", расположенной вместо обычной вкладки "результаты поиска", добавляются имя, фамилия и отчество студента, чтобы его можно было идентифицировать в списке; затем, выбрав этого студента из списка уже в качестве его атрибутов в приказе можно задать ему все остальные параметры, в том числе - группу, курс, пол и т.п.; при введении в действие такого приказа в БД будет создана реальная запись об обучающемся и в дальнейшем с ним можно будет работать всеми обычными средствами;
