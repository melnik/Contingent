Список приказов

Рабочее пространство разделено на следующие части:

  • поиск — инструмент для формирования списка приказов;
  • список приказов, включающий, в свою очередь, следующие части:
    • панель инструментов — содержит кнопки для работы с приказами;
    • заголовок списка — содержит названия колонок списка;
    • тело списка — содержит набор строк, в каждой из которых представлена краткая информация о конкретном приказе;
    • строка состояния — отображает количество приказов в списке и, возможно, другую статисику.

Возможные действия

  • Получение списка приказов — воспользуйтесь поиском, расположенным в левой части экрана. После этого в теле списка появится краткая информация о приказах, соответствующих критериям указанным при поиске.
  • Просмотр / редактирование приказа — нажмите на строчку, соответствующую тому приказу, который Вы хотели бы посмотреть или отредактировать. В зависимости от Ваших прав в системе и состояния приказа (проект / в стадии подписания / введен в действие) откроется окно редактирования приказа, либо — если редактировние невозможно — Вам будет предоставлен его печатный вариант или откроется окно ввода приказа в действие.
  • Создание проекта приказа — нажмите на кнопку , расположенную в панели инструментов. После этого откроется окно создания проекта приказа.
