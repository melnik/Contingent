Список студентов

Список студентов разделен на следующие части:

  • панель закладок — содержит закладки с названиями разделов (формулировок) приказа, каждой из которых соответствует список студентов, относящихся к данному разделу, и, как правило, отдельную закладку, которая служит для добавления студентов в приказ и, в зависимости от типа приказа, может содержать:
    • "Результаты поиска" — результаты поиска или навигации — для приказов, в которых может фигурировать произвольное количество студентов;
    • "Остальные" — студенты, которые должны фигурировать в приказе, но по каким-либо причинам в нем отсутствуют — для приказов, относящихся к какому-либо из структурных подразделений Университета (например, курсу в рамках одного факультета);
    • "Добавление студента" — форма для добавления в приказ студента, информация о котором пока отсутствует в базе — для приказов о зачислении.
  • панель инструментов (находится в строке состояния - в правом нижнем углу) — содержит кнопки для работы со студентами в приказе;
  • заголовок списка — содержит названия колонок списка и кнопку (), выделяющую / снимающую выделение со всех студентов в приказе;
  • тело списка — содержит набор строк, в каждой из которых представлена краткая информация о конкретном студенте, и кнопку (), служащую для выбора данного студента;
  • строка состояния — отображает количество студентов в списке и другую статисику (например, распределение студентов по основам обучения, половому признаку и пр.).

Возможные действия

  • Получение списка студентов для последующего добавления их в приказ — воспользуйтесь поиском или навигацией, расположенными в левой части экрана. После этого в теле списка (закладка "Результаты поиска") появится краткая информация о студентах, соответствующих критериям указанным при поиске или выбранному структурному подразделению Университета. При этом в список попадают только те обучающиеся, которые могут фигурировать в данном приказе.
  • Просмотр / редактирование атрибутов приказа, относящихся к студенту — нажмите на строчку, соответствующую тому студенту, информацию о котором Вы хотели бы посмотреть или отредактировать. Атрибуты приказа, относящиеся к данному студенту, будут отображены в окне редактирования атрибутов приказа, относяшихся к конкретному студенту, расположенном над списком.
  • Просмотр / редактирование атрибутов группы, относящихся к студенту — нажмите на строчку, соответствующую той группе, информацию о которой Вы хотели бы посмотреть или отредактировать. Атрибуты приказа, относящиеся к данной группе, будут отображены в окне редактирования атрибутов приказа, относяшихся к конкретной группе, расположенном над списком.
  • Добавление студентов в приказ — воспользуйтесь одной из закладок "Результаты поиска" или "Остальные" для получения списка студентов, которые могут быть добавлены в приказ, либо закладкой "Добавление студента" — для добавления в приказ нового студента (доступность этих закладок зависит от типа приказа).
