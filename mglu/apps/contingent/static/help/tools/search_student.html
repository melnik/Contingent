Поиск

Инструмент "Поиск" позволяет проводить поиск обучающихся по одному или нескольким критериям сразу. В качестве критериев могут выступать: фамилия, имя, отчество, пол, группы, номер личного дела, состояние, форма (основа) обучения, проживание в общежитие. Каждое из этих полей может быть либо оставлено пустым (в таком случае это поле не участвует в качестве критерия), либо содержать некую запись и тогда будут отбираться только обучающиеся, у которых заданный параметр соответствует записи. По текстовым полям поиск может происходить по начальным символам, то есть по запросу Ф.И.О. = "Иван" будут находиться фамилии "Иванов", "Иванова", "Иваненко", "Иванин" и т.п.

В зависимости от контекста, в котором используется инструмент, список найденных обучающихся появляется в соответствующем окне. Таким образом, при использовании инструмента в режимах:

  • Студенты - список студентов появляется справа; если у вас достаточно привилегий для просмотра или редактирования карточки студента - указав студента в этом списке вы можете перейти к его карточке.
  • Приказы - список студентов появится во вкладке "остальные" среди всех списков студентов. Отобранных студентов можно будет пометить и перенести в другие вкладки, таким образом, добавив их в тот или иной параграф приказа.
