Поиск приказов

Инструмент "Поиск" позволяет проводить поиск приказов по одному или нескольким критериям сразу. В качестве критериев могут выступать:

  • номер — номер приказа, присвоенный ему при редактировании или введении в действие;
  • факультет — аббревиатура факультета, издавшего приказ;
  • тип — тип приказа (сокращенное наименование);
  • статус — статус приказа (проект / введен в действие);
  • дата — диапазон дат, задающий — в зависимости от статуса приказа — дату создания проекта, если приказ не введен в действие, или дату введения приказа в действие — в противном случае;

Каждое из этих полей может быть либо оставлено пустым — в таком случае поиск по этому полю не проводится, либо содержать некую запись и тогда будут отбираться только те приказы, у которых заданный параметр соответствует этой записи. По текстовым полям (тем, в которые текст набивается вручную) поиск может происходить по префиксу (нескольким начальным символам), то есть по запросу номер = "18" могут быть найдены приказы с номерами 18-20/32ИУ, 18-13/26МТ и т. п.

Поиск ведется без учета регистра (строчные / заглавные буквы), то есть запросы номер = "18-20/32ИУ" и номер = "18-20/32иу" абсолютно идентичны.

Поиск производится по нажатию на кнопку . После этого результаты поиска (список приказов) появятся в окне справа.
